## Overview

This repository contains Helm charts for deploying [Contour](https://projectcontour.io/), an open-source Kubernetes ingress controller that uses [Envoy](https://www.envoyproxy.io/) as a reverse proxy and load balancer.

## Charts

| Chart | Description |
|-------|-------------|
| [contour](charts/contour) | Deploys Contour ingress controller and Envoy proxy |
